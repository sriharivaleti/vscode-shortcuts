---

<h3 style={{width: '95%', textAlign: 'center'}}>Why This Translation?</h3>

<p style={{width: '95%', textAlign: 'center'}}>When I first came across the VSCode shortcut cheat sheet, I couldn't make out any of their <code>gobbledy-gook</code>. I'm referring to the shortcut symbols that appeared everywhere on the sheet with no explanation of what each represented.</p>

---

<p style={{width: '95%', textAlign: 'center'}}>I put together a translation of all the shortcuts represented in the VSCode cheat sheet. Enjoy!</p>

---

<h3 style={{width: '95%', textAlign: 'center'}}>General</h3>

<code style={{fontSize: '2rem', width: '95%', background: '#fff', padding: '5px 10px'}}>
    <span style={{color: '#cc0033'}}>Shift + Command + P || F1 -</span> Show Command Palette<br/>
    <span style={{color: '#cc0033'}}>Command + P -</span> Quick Open, Go to File<br/>
    <span style={{color: '#cc0033'}}>Shift + Command + N -</span> New Window/Instance<br/>
    <span style={{color: '#cc0033'}}>Command + W -</span> Close Window/Instance<br/>
    <span style={{color: '#cc0033'}}>Command + , -</span> User Settings<br/>
    <span style={{color: '#cc0033'}}>Command + K Command + S -</span> Keyboard Shortcuts
</code>

---

<h3 style={{width: '95%', textAlign: 'center'}}>Basic Editing</h3>

<code style={{fontSize: '2rem', width: '95%', background: '#fff', padding: '5px 10px'}}>
    <span style={{color: '#cc0033'}}>Command + X -</span> cut line (empty selection)<br/>
    <span style={{color: '#cc0033'}}>Command + C -</span> Copy Line (empty selection)<br/>
    <span style={{color: '#cc0033'}}>Option + Down Arrow/Option + Up Arrow -</span> Move line down/up<br/>
    <span style={{color: '#cc0033'}}>Shift + Option + Down Arrow/Shift + Option + Up Arrow -</span> Copy line down/up<br/>
    <span style={{color: '#cc0033'}}>Shift + Command + K -</span> Delete line<br/>
    <span style={{color: '#cc0033'}}>Command + Enter/ Shift + Command + Enter -</span> Insert line down/up<br/>
</code>

---

<h3 style={{width: '95%', textAlign: 'center'}}>Basic Editing Contd</h3>

<code style={{fontSize: '2rem', width: '95%', background: '#fff', padding: '5px 10px'}}>
    <span style={{color: '#cc0033'}}>Shift + Command + \ -</span> Jump to matching bracket<br/>
    <span style={{color: '#cc0033'}}>Command + ]/Command + [ -</span> Indent/Outdent line<br/>
    <span style={{color: '#cc0033'}}>Home / End -</span> Go to beginning/end of line.<br/>
    <span style={{color: '#cc0033'}}>Command + Up Arrow -</span> go to the beginning of the file.<br/>
    <span style={{color: '#cc0033'}}>Command + Down Arrow -</span> go to the end of the file.<br/>
    <span style={{color: '#cc0033'}}>Option + Command + [ / Option + Command + ] -</span> Fold/Unfold region.<br/>
</code>

<h3 style={{width: '95%', textAlign: 'center'}}>Basic Editing Contd</h3>

<code style={{fontSize: '2rem', width: '95%', background: '#fff', padding: '5px 10px'}}>
    <span style={{color: '#cc0033'}}>Command + K + Command + [ / Command + K + Command + ] -</span> Fold/Unfold all sub-regions.<br/>
    <span style={{color: '#cc0033'}}>Command + K + Command + 0 / Command + K + Command + J -</span> Fold/Unfold all regions.<br/>
    <span style={{color: '#cc0033'}}>Command + K + Command + C -</span> Add line comment.<br/>
    <span style={{color: '#cc0033'}}>Command + K + Command + U -</span> Remove line comment.<br/>
    <span style={{color: '#cc0033'}}>Command + / -</span> Toggle line comment.<br/>

    <span style={{color: '#cc0033'}}>Option + Z -</span> Toggle word wrap
</code>

<h3 style={{width: '95%', textAlign: 'center'}}>Multi-Cursor and Selection</h3>

<code style={{fontSize: '2rem', width: '95%', background: '#fff', padding: '5px 10px'}}>

    <span style={{color: '#cc0033'}}>Option + Comment + Down Arrow -</span> Insert cursor below.<br/>

</code>
