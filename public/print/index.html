Visual Studio Code Shortcuts

A translation of the Visual Studio Code Keyboard Shortcuts for macOS Cheat Sheet

Why This Translation?

When I first came across the VSCode shortcut cheat sheet, I couldn't make out any of their gobbledy-gook. I'm referring to the shortcut symbols that appeared everywhere on the sheet with no explanation of what each represented.

I put together a translation of all the shortcuts represented in the VSCode cheat sheet. Enjoy!

General

Shift + Command + P || F1 - Show Command Palette
Command + P - Quick Open, Go to File
Shift + Command + N - New Window/Instance
Command + W - Close Window/Instance
Command + , - User Settings
Command + K Command + S - Keyboard Shortcuts

Basic Editing

Command + X - cut line (empty selection)
Command + C - Copy Line (empty selection)
Option + Down Arrow/Option + Up Arrow - Move line down/up
Shift + Option + Down Arrow/Shift + Option + Up Arrow - Copy line down/up
Shift + Command + K - Delete line
Command + Enter/ Shift + Command + Enter - Insert line down/up

Basic Editing Contd

Shift + Command + \ - Jump to matching bracket
Command + ]/Command + [ - Indent/Outdent line
Home / End - Go to beginning/end of line.
Command + Up Arrow - go to the beginning of the file.
Command + Down Arrow - go to the end of the file.
Option + Command + [ / Option + Command + ] - Fold/Unfold region.

Basic Editing Contd

Command + K + Command + [ / Command + K + Command + ] - Fold/Unfold all sub-regions.
Command + K + Command + 0 / Command + K + Command + J - Fold/Unfold all regions.
Command + K + Command + C - Add line comment.
Command + K + Command + U - Remove line comment.
Command + / - Toggle line comment.
Shift + Option + A - Toggle block comment.
Option + Z - Toggle word wrap

Multi-Cursor and Selection

Option + click - Insert cursor.
Option + Command + Up Arrow - Insert cursor above.
Option + Comment + Down Arrow - Insert cursor below.
Command + U - Undo last cursor operation.
Shift + Option - Insert cursor at end of each line selected.

Multi-Cursor and Selection Contd

Command + L - Select current line.
Shift + Command + L - Select all occurrences of current selection.
Command + F2 - Select all occurrences of current word.
Control + Shift + Command + Right Arrow/Left Arrow - Expand/Shrink selection.
Shift + Option + Drag Mouse - Column (box) selection.

Multi-Cursor and Selection Contd

Shift + Option + Command + Up Arrow/Down Arrow - Column (box) selection up/down
Shift + Option + Command + Left Arrow/Right Arrow - Column (box) selection left/right
Shift + Option + Command + PgUp - Column (box) selection page up.
Shift + Option + Command + PgDn - Column (box) selection page down.

Search and Replace

Command + F - Find
Option + Command + F - Replace
Command + G/Shift + Command + G - Find next/previous.
Option + Enter - Select all occurrences of Find match
Command + D - Add selection to next Find match
Command + K + Command + D - Move last selection to next Find match

Rich Languages Editing

Control + Space - Trigger suggestion
Shift + Command + Space - Trigger parameter hints
Shift + Option + F - Format document
Command + K + Command + F - Format selection
F12 - Go to definition
Command + K + F12 - Open definition in the side

Rich Languages Editing Contd

Command - Quick Fix
Shift F12 - Show References
F2 - Rename Symbol
Command + K + Command + X - Trim trailing whitespace
Command + K + M - Change file language

Navigation

Command + T - Show all Symbols
Control + G - Got to Line
Command + P - Go to file
Shift + Command + O - Go to Symbol

Navigation Contd

Shift + Command + M - Show Problems panel
F8/Shift + F8 - Go to next/previous error or warning
Control + hyphen/Control + Shift + hyphen - Go back/go forward
Control + Shift + M - Toggle Tab moves focus

Editor Management

Command + W - Close editor
Command + K + F - Close Folder
Command + \ - Split editor
Command + 1 / Command + 2 / Command + 3 - Focus into 1st, 2nd, 3rd editor group.

Editor Management Contd

Command + K + Command + Left Arrow / Command + K + Command + Right Arrow - focus into previous/next editor group.
Command + K + Shift + Command + Left Arrow / Command + K + Shift + Command + Right Arrow - Move editor left/right
Command + K + Left Arrow / Command + K + Right Arrow - Move active editor group

File Management

Command + N - New file
Command + O - Open file
Command + S - Save
Shift + Command + S - Save as
Option + Command + S - Save All
Command + K + Command + W - Close All

File Management Contd

Shift + Command + T - Reopen closed editor
Command + K + Enter - Keep preview mode editor open
Control + Tab / Control + Shift + Tab - Open next/previous
Command + K + P - Copy path of active file
Command + K + R - Reveal active file in Finder
Command + K + O - Show active file in new window/instance

Display

Control + Command + F - Toggle full screen
Option + Command + 0 - Toggle editor layout (horizontal/vertical)
Command + Equal/Shift + Command + Hyphen - Zoom in/out
Command + B - Toggle Sidebar visibility
Shift + Command + E - Show Explorer/Toggle focus

Display Contd

Shift + Command + F - Show Search
Control + Shift + G - Show Source Control
Shift + Command + D - Show Debug
Shift + Command + X - Show Extensions
Shift + Command + H - Replace in files

Display Contd

Shift + Command + J - Toggle Search details
Shift + Command + U - Show Output Panel
Shift + Command + V - Open Markdown preview
Command + K + Z - Zen Mode (Esc Esc to exit)

Debug

F9 - Toggle Breakpoint
F5 - Start/Continue
F11/Shift + F11 -Step into/out
F10 - Step over
Shift + F5 - Stop
Command + K + Command + I - Show hover

Integrated Terminal

Control + backtick - Show Integrated Terminal
Control + Shift + backtick - Create new Terminal
Command + C - Copy selection
Command + Up Arrow / Down Arrow - Scroll up/down
PgUp/PgDn - Scroll page up/down
Command + Home / End - scroll to top/bottom
